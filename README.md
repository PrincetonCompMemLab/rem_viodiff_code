# REM_VIODIFF_code

Task and analysis code for McDevitt, E. A., Kim, G., Turk-Browne, N. B. & Norman, K. A. (2025). The role of rapid eye movement sleep in neural differentiation of memories in the hippocampus. Journal of Cognitive Neuroscience, 10.1162/jocn.a.82.

Related neuroimaging data are on OpenNeuro.org: 
https://openneuro.org/datasets/ds006576
